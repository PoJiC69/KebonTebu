# KEBONTEBU Casino — Demo (Flutter)

Ini contoh proyek demo cross-platform (Android, iOS, Web) menggunakan Flutter. Tujuannya: kerangka UI/UX modern & elegan, dan screen demo untuk game lokal (non‑real‑money).

Fitur yang disertakan:
- Tema modern dan grid home screen.
- Demo lokal untuk: Poker, Qiu Qiu, Samgong, Togel 12-digit.
- Struktur mudah dikembangkan untuk menambahkan server real-time, game engine, dan integrasi backend.

Penting — legal & compliance:
- Contoh ini tidak mengandung mekanisme pembayaran, ledger, atau fitur real‑money. Untuk menjalankan kasino online yang melibatkan uang sungguhan Anda harus:
  - Memastikan lisensi perjudian di yurisdiksi target.
  - Mematuhi peraturan KYC/AML, perlindungan pemain, dan pajak.
  - Menyediakan backend aman untuk transaksi, audit, dan anti‑fraud.

Instalasi cepat:
1. Pastikan Flutter SDK terpasang (https://flutter.dev).
2. Clone / buat project baru dan ganti file sesuai contoh.
3. Jalankan:
   - flutter pub get
   - flutter run -d chrome (untuk web)
   - flutter run (untuk device Android/iOS)

Langkah berikut yang saya rekomendasikan:
- Tambahkan state management dan arsitektur backend (WebSocket server) untuk multiplayer.
- Implementasikan engine permainan yang sesuai aturan resmi (hand ranking, odds).
- Integrasi audit log, database, dan server otentikasi.
- Desain aset grafis dan animasi premium (SVG/Flare/Rive).

Jika Anda ingin, saya bisa:

- Menambahkan flow otentikasi & mock wallet (non‑real money),

Beritahu langkah mana yang ingin Anda lanjutkan dan yurisdiksi tempat aplikasi ini akan dijalankan (untuk pertimbangan legalisasi).
